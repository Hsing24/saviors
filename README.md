# Saviors

A Chrome extension project.

## 簡介

此專案為 Chrome extension，無需連接資料庫、無需後端邏輯。

## 安裝說明

1. 開啟 Chrome 瀏覽器
2. 前往 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇此專案資料夾

## 專案結構

```
saviors/
├── manifest.json    # Chrome extension 配置檔
├── popup.html       # 彈出視窗 HTML
├── popup.js         # 彈出視窗 JavaScript
├── popup.css        # 彈出視窗樣式
├── icons/           # 擴充功能圖示
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── LICENSE
└── README.md
```
